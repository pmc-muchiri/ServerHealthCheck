# Housekeeping Desktop

PySide6 desktop version of the System Housekeeping dashboard.

## Run

```powershell
cd e:\pmc\projects\Housekeeping\housekeeping_desktop
python main.py
```

## Notes

- The app inspects the local machine by default.
- You can enter a remote Windows server name or IP and run the same inspection logic.
- Remote inspection depends on PowerShell / CIM access and the current user's permissions.
